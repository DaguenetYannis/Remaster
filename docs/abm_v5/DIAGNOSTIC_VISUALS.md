# ABM_v5 Diagnostic Visuals

## Purpose

These visuals are a post-Phase-2 diagnostic layer for inspecting the empirical
terrain built by the accounting, capability, network, and phase-space panels.
They are labeled as Phase 2 diagnostic exploration because they are not model
results, regimes, thresholds, or scenario outcomes.

## Focus

The diagnostics focus on two questions:

- how network position shapes observed country-sector nodes;
- how observed phase space reveals tension between capability and constraint.

The main contrasts are local green-ness versus network green exposure, carbon
centrality versus network exposure, supplier lock-in versus green capability,
emissions-intensity gap versus green capability, and brown centrality versus
green capability.

## Interpretation Boundaries

These figures are descriptive. They do not assign regimes, discover thresholds,
cluster nodes, infer behavioural rules, simulate dynamics, or make causal
claims. Supplier candidates are treated as opportunity structure only, not as
supplier choices or rewiring decisions.

## Main Visuals

- `local_greenness_vs_network_green_exposure.png` separates local emissions
  position from embedded network exposure.
- `brown_centrality_vs_network_green_exposure.png` shows whether central carbon
  positions sit in greener or browner production neighborhoods.
- `supplier_lock_in_vs_green_capability.png` contrasts constraint and
  transition potential.
- `emissions_intensity_gap_vs_green_capability.png` highlights dirty-but-capable
  and clean-but-less-capable nodes.
- `brown_centrality_vs_green_capability.png` maps strategic transition terrain.
- `selected_node_trajectories_ei_gap_vs_network_green_exposure.png` traces a
  small automatically selected set of illustrative nodes through observed
  phase space.

## Design-Target Placeholders

Design-target variables with zero availability are not plotted as observed
axes. They remain explicit missing placeholders so future phases can add them
without confusing unavailable variables with empirical evidence.
